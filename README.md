# Iris Flower Classification using Machine Learning

This project is developed as part of the Oasis Infobyte Internship (OIBSIP).

## Project Objective

The main objective of this project is to build a Machine Learning model that can classify Iris flowers into three species based on their measurements.

The three Iris flower species are:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

---

# Dataset Information

The dataset contains the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Variable:
- Species

The following Machine Learning algorithms were implemented:
